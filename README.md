# How to run this project

1. Clone this repository
   ```shell
     git clone git@github.com:jeeehaan/LokerDesain.git
   ```
2. Install dependencies
   ```shell
     npm install
   ```
3. Go to project folder
   ```shell
     cd LokerDesain
   ```
4. Run the project
   ```shell
     npm run dev
   ```
